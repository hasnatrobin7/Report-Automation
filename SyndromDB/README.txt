SyndromDB Folder Structure Instructions
====================================

For each unique syndrom (failure type), create a subfolder named after the syndrom.

Inside each subfolder, add:
  - golden.jpg      (Golden/reference image for this syndrom)
  - defect.jpg      (Defect/failure image for this syndrom)
  - description.txt (A text file describing the syndrom/failure)
